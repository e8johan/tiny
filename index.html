What is tiny-js?

Tiny-js is a minimalist, Linux-based, Javascript platform. It provides the basis for a Linux distribution that when built for x86 (qemu) has needs the following resources:

  • 3.5M root filesystem.
  • 2.2M kernel.
  • 8M of RAM required.

It is, of course, platform independent and can be built for any architecture supported by the Linux kernel.
